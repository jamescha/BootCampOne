The declaration must be the very first thing in your HTML document. The <html> tag tells the browser that this is an HTML document. The <head> element is a container for all the head elements. The <title> tag defines the title of the document. The <body> tag defines the document's body. The <h1> to <h6> tags are used to define HTML headings. The <p> tag defines a paragraph. The <ul> tag defines an unordered (bulleted) list. The <ol> tag defines an ordered list. The <li> tag defines a list item. Will pull what is currently on your Github repository.
