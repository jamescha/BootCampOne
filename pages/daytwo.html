The <h1> to <h6> tags are used to define HTML headings. The <p> tag defines a paragraph. The <ul> tag defines an unordered (bulleted) list. The <ol> tag defines an ordered list. The <li> tag defines a list item. The <div>tag defines a division or a section in an HTML document. The <header> tag specifies a header for a document or section. The <footer> tag defines a footer for a document or section. The content attribute gives the value associated with the http-equiv or name attribute.
